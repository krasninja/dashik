The list of currently available software with the similar functionality.

## Active

1. [Glance](https://github.com/glanceapp/glance). A lightweight, highly customizable dashboard that displays your feeds in a beautiful, streamlined interface. Go.

2. [YASB](https://github.com/amnweb/yasb). A highly configurable Windows status bar written in Python.

3. [Polybar](https://github.com/polybar/polybar). A fast and easy-to-use status bar.

4. [BeWidgets](https://apps.microsoft.com/detail/9nq07fg50h2q).

5. [Widgets Box](https://apps.microsoft.com/detail/9nf1zhfjbczj).

6. [GadgetPack](https://gadgetpack.net/index.html).

7. [Widget Launcher](https://apps.microsoft.com/detail/9wzdncrdqfbt)

8. [Dashy](https://dashy.to). Dashy is an open source, highly customizable, easy to use, privacy-respecting dashboard app. Vue.

## Abondened

1. [Tabame](https://github.com/Far-Se/tabame). Open source. The little widget that has many features.

2. [Linkbar](https://github.com/ATGH15102AFMLD/Linkbar). Linkbar is a free source code desktop toolbar.

3. [candybar](https://github.com/Lokaltog/candybar). candybar - WebKit-based status bar for tiling window managers.

4. [lemonbar](https://github.com/LemonBoy/bar). lemonbar - Featherweight lemon-scented bar.

5. [tint2](https://gitlab.com/o9000/tint2). A lightweight panel/taskbar for Linux and BSD.

6. [XMeters](https://entropy6.com/xmeters/).

7. [DevHome](https://github.com/microsoft/devhome). Dev Home is a new control center for Windows providing the ability to monitor projects in your dashboard using customizable widgets, set up your dev environment by downloading apps, packages, or repositories, connect to your developer accounts and tools (such as GitHub), and create a Dev Drive for storage all in one place. Archived.
